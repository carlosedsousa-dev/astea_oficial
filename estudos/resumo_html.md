# Síntese de Estudo: HTML

## O que é HTML?

O **HTML (HyperText Markup Language)** não é uma linguagem de programação. Ele é uma **linguagem de marcação**, criada para organizar e descrever o conteúdo de páginas na web.

Diferente de linguagens que executam cálculos ou controlam fluxos lógicos, o HTML define **a estrutura da informação**, indicando ao navegador o que é título, texto, imagem ou link.

Pode-se dizer que ele representa a base estrutural de um site, servindo como a fundação sobre a qual o restante é construído.

---

## Principais Tags e Suas Funções

As *tags* compõem a sintaxe do HTML e informam ao navegador o significado de cada trecho do conteúdo.

- **`<h1>` até `<h6>`**: Representam **níveis de cabeçalho**, organizando o conteúdo por ordem de importância. O `<h1>` é o nível mais alto, enquanto os demais indicam subdivisões.
- **`<p>`**: Utilizada para delimitar um **bloco de texto** em formato de parágrafo.
- **`<a>`**: Elemento responsável por criar **hiperlinks**. Utiliza o atributo `href` para definir o endereço de destino.
- **`<img>`**: Insere **imagens** na página. Não exige tag de fechamento e utiliza `src` (origem da imagem) e `alt` (descrição alternativa).
- **`<ul>` e `<ol>`**: Estruturam **listas**. A `<ul>` gera listas com marcadores, enquanto a `<ol>` cria listas numeradas.
- **`<li>`**: Define cada **entrada** dentro de uma lista.
- **`<strong>` ou `<b>`**: Aplicam **destaque em negrito** ao texto.
- **`<em>` ou `<i>`**: Indicam **ênfase**, normalmente exibida em itálico.
- **`<br>`**: Insere uma **quebra de linha** simples.

---

## Estruturação com a Tag `<div>`

A tag **`<div>`** (de *division*) é amplamente utilizada para **agrupar elementos** dentro de uma mesma seção da página.

Ela atua como um contêiner genérico que permite organizar o conteúdo em blocos. Esse agrupamento facilita o chamado **aninhamento**, que consiste em colocar elementos dentro de outros elementos.

Essa prática contribui para uma estrutura mais clara, melhora a manutenção do código e possibilita aplicar estilos e comportamentos de forma organizada posteriormente.
